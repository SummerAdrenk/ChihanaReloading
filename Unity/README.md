# **UnityTools**
> - Please only use UnityFS.py, arc_unpack.py, and arc_pack.py; ignore the rest.
> - arc_unpack.py and arc_pack.py were modified from SE's Unity project and now support batch processing.

# **♯ arc_unpack.py & arc_pack.py**
> - As the names suggest, these are unpacking and packing programs designed for .arc archives with the @ARCH000 file header.
> - Please note: When packing, the timestamp file extracted during the unpacking process must be placed inside the folder to be packed.

# **♯ UnityFS.py**
> - A program for processing UnityFS files; please refer to the code for details.
> - Currently, the repacking of VideoClips has not yet been resolved.
